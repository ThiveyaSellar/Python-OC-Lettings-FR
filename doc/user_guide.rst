User's guide
===============


Use Case 1: Browse Available Properties
--------------------------------------

1. The user opens the application homepage at `http://127.0.0.1:8000/`.
2. From the homepage, the user navigates to the “Properties” section by clicking on the “Lettings” button in the page or in the navigation bar.
3. The user sees a list of available properties with their titles and brief details.
4. By clicking on a property title, the user accesses the detailed page for that property.
5. The user can view the full address of the property.

Use Case 2: View User Profiles
------------------------------

1. The user opens the application homepage at `http://127.0.0.1:8000/`.
2. The user browses the list of user profiles.
3. By selecting a username, the user navigates to the detailed profile page.
4. The user can see information linked to that profile, such as name, email and favorite city.
